## mGBA Configuration

mGBA allows path changes, so just change the paths in Tools>Settings>Paths and create inside the Syncthing folder, a folder for each path (saves, states, screenshots...). You can just make a save folder if you don't generally do save states or screenshots, that's something personal. 

Change the default path to ```/home/user/Syncthing folder/GBA/```, and mGBA will store the ```.sav``` files there. 

So mGBA is also one of the most straightforward emulators to configure.
